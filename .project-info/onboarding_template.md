## Setup trong 5 phút

```bash
# 1. Clone & install
git clone <repo>
cd <project>
npm install

# 2. Setup env
cp .env.example .env
# Điền các values cần thiết

# 3. Setup database (nếu có)
npm run db:migrate

# 4. Chạy dev
npm run dev
```
