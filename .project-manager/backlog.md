```markdown

**Priority:** P0/P1/P2/P3
**Epic:** Epic name
**Description:** What needs to be built
**Acceptance Criteria:**
- [ ] Criterion 1
- [ ] Criterion 2

**Design Spec:** (link to Atlassian or docs/)
**Dev Status:** Open / In Progress / Done
```

---

## Getting Started

### Business Process

1. **Create task** in backlog.md (or Atlassian Backlog MCP)
2. **Design/Spec** — document requirements
3. **Handoff** — create design document in `.design-handoff/`
4. **Track** — monitor dev progress

### Handoff to Developers

When task is ready for development:
1. Finalize spec in Atlassian
2. Copy to `docs/` or `.design-handoff/`
3. Update dev branch with link to design
4. Run `/handoff` command
