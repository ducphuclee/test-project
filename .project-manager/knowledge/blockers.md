# Blockers

Các vấn đề đang chặn tiến độ. Cập nhật khi resolve.

---

## Format

```markdown
## [BLOCK-001] Tên blocker
**Reported:** YYYY-MM-DD by @agent
**Affects:** TASK-001, TASK-002
**Status:** Open / In Investigation / Resolved
**Description:** Mô tả vấn đề
**Attempted:** Những gì đã thử
**Resolution:** (điền khi resolved)
```
